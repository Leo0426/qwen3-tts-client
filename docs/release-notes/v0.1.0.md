macOS 原生 Qwen3-TTS 本地推理客户端首个版本。完全离线，无需 API Key。

## 亮点

- **本地流式合成**：MLX 推理，M 系列芯片首包约 0.1 秒，边生成边播放
- **三种发声方式**：9 个官方预置音色（含方言）· 声音克隆（3 秒参考音频）· 语音设计（文字描述造声音，支持试听迭代）
- **风格指令**：「用温柔的语气慢慢说」级别的情感控制
- **长文本**：句子边界自动分段，整篇文章连续播放；真实音频波形，点按跳转，0.5×~2× 变速
- **系统集成**：菜单栏常驻，任意 App 内 ⌘C + ⌃⌥⌘S 即刻朗读
- **模型管理**：下载中心（官方/镜像/自定义源、自定义存储路径）+ 内存加载管理

## 系统要求

Apple Silicon（M1+）· macOS 14+ · 每模型磁盘 1.8~3 GB、内存约 2 GB

## 安装

下载 zip 解压，拖入「应用程序」。应用为 ad-hoc 签名，首次打开请右键 →「打开」，
或 `xattr -cr "/Applications/Qwen3 TTS.app"`。

首次启动引导下载模型，此后完全离线。
