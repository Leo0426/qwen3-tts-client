修复与增强版本。完全离线的 macOS 原生 Qwen3-TTS 客户端。

## 新增

- **m4a 导出**：WAV 之外新增 AAC (.m4a) 格式，文件小一个量级；历史条目导出同样支持
- **自定义全局快捷键**：设置 → 快捷键，可在 ⌃⌥⌘S / ⌃⌥S / ⌃⌥⌘R / ⇧⌘Y 间切换或禁用，注册冲突有提示
- **缺模型引导**：所需模型未下载时，错误提示旁直接给「去下载」按钮打开下载中心

## 修复

- 语音设计弹窗的「试听/停止」按钮在音频播放结束后不复位
- 错误提示现在可手动关闭（此前会一直挂到下一次合成）

## 工程

- MIT 许可证
- GitHub Actions CI：PR/push 跑测试，推 v* 标签自动构建发布
- 打包脚本一条命令产出 .app + zip + sha256，版本号取自 git tag
- 新增 AppModel 状态机测试与 m4a 编码测试（25 个测试）

## 安装

下载 `Qwen3TTS-v0.1.1-arm64.zip` 解压，拖入「应用程序」。应用为 ad-hoc 签名，
首次打开请**右键 →「打开」**，或运行 `xattr -cr "/Applications/Qwen3 TTS.app"`。

系统要求：Apple Silicon（M1+）· macOS 14+
